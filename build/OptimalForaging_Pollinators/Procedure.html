User Instructions:


 The simulator consists of three regions: the simulator's viewable window, the variables menu and the simulation control menu.


The Variables Menu:


User can enter the values for the different variables in the simulator. There is one more button ‘More Variables’ is there in the variables menu. By clicking on this button, we will get a ‘more variables’ window (as shown in the figure), which contains some additional parameters too.

 

The variable menu consists of text boxes to enter the values of number of calories and value of Y-intercept. There is also slider bars to change the values of the variables such as, Number of position in flowers probability that the plant has not been drained of nectar (P), Energy spent in flying from one plant to another (eb) and time spent in flying from one plant to another. By clicking on the ‘more variables’ button, you will get some additional parameters. Move the pointer across the slider bar to know the names of the variables.

 

Simulation control menu:


These set of buttons are used for controlling the simulation. Simulation control includes: buttons for plotting Net Calorie, Total Energy and 'Reset' button.

 

Result display window:


The simulated values for the parameters are interspersed as graphs in this window.
